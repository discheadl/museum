Coloca aqui tus fotos y videos si quieres servirlos desde Express.

Ejemplo:
- `api/public/media/salas/origen/vasija.jpg`
- `api/public/media/salas/origen/recorrido.mp4`

Luego cambia las URLs del archivo `api/src/data/museum-data.js` para apuntar a:
- `http://localhost:4000/media/salas/origen/vasija.jpg`
- `http://localhost:4000/media/salas/origen/recorrido.mp4`
